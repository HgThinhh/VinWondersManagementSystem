# VinWonders Management System

## Overview

VinWonders Management System is a desktop application developed using C#, Windows Forms, SQL Server, Stored Procedures, and Crystal Reports following a Three-Tier Architecture (GUI, BUS, DAL).

The system is designed to support the management and daily operations of a theme park by providing modules for ticket management, customer management, employee scheduling, store operations, maintenance tracking, and business reporting.

The project aims to improve operational efficiency, simplify data management, and provide real-time insights through dashboards and statistical reports.

## Technologies

* C#
* Windows Forms
* SQL Server
* Stored Procedures
* Crystal Reports
* Three-Tier Architecture (GUI - BUS - DAL)
* Git & GitHub

## Key Features

### Dashboard

* Overview of operational activities
* Quick access to management modules
* Business statistics and monitoring

### Ticket Management

* Ticket type management
* Ticket sales and invoice generation
* Invoice history tracking
* Crystal Reports integration

### Employee Management

* Employee information management
* Shift scheduling
* Position and role management

### Customer Management

* Customer information management
* Membership level management
* Customer tracking and support

### Store & Product Management

* Product inventory management
* Sales processing
* Invoice generation
* Product stock monitoring

### Maintenance Management

* Maintenance request creation
* Equipment maintenance tracking
* Repair progress monitoring
* Maintenance report generation
